# Ozempic YouTube Data Scraping & Analysis |

This project collects and analyzes public YouTube data around Ozempic, a diabetes medication that has become a cultural phenomenon extending far beyond its original medical purpose. It involves fetching video metadata and comments 
via the YouTube Data API, extracting URLs shared in comments, and enriching the dataset by scraping linked web pages. Analysis covers video engagement patterns, most liked comments, external link domains, and influential commenters.

Tools: Requests, BeautifulSoup, Pandas, Seaborn, YouTube Data API v3.
